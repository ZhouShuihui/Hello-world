##  rust表达式和语句
----------------------------
  表达式会求值并返回值，语句不返回值，语句中可以包含表达式
-  函数调用是一个表达式,函数定义是语句
-  宏调用是一个表达式
-  创建作用域的代码块是一个表达式,若代码块里没有表达式，即全为语句或为空代码块，则返回空元组()
---------------------------------

```rust
 // 非法赋值，因为 let y = 0 为语句，无返回值
let x = （let y = 0);
// 以下{}代码块为表达式
let k = { 
    let x = 3;
    x + 1
    };
```
